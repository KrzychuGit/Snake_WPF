  <p>
  In this game, we control the snake, and eat apples thanks to which the snake grows and we get points. </br>
  If the snake eats poison or bites itself or the opponent, the player loses.</br>
  The player who scored the most points is saved.
 </p>
<p>
 This is a game with one and two players mode.
